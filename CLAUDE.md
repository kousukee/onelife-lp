# ONELIFE LP - プロジェクト指示書

## 概要
ONELIFEアプリのランディングページ。GitHub Pagesでホスティング中。

## ファイル構成
```
onelife-lp/
  index.html       # LP本体（1ファイル完結）
  ロゴ_ONELIFE.png  # アプリロゴ（ChatGPTで生成）
```

## 技術
- 純粋なHTML/CSS（フレームワークなし）
- Google Fonts: Nunito（ロゴテキスト用）
- ホスティング: GitHub Pages（公開済み）

## デザイン方針
- アプリのカラーパレットに統一（ONELIFE本体と同じ）
- プライマリ: #8B6F47 / セカンダリ: #E8A87C / アクセント: #5B8C5A
- レスポンシブ対応済み（スマホ・タブレット・PC）

## LP構成
1. ナビ（ロゴ + ダウンロードボタン）
2. ヒーロー（キャッチコピー + App Storeボタン）
3. モックアップ4画面（SVGで作成）
4. 共感ゾーン
5. 機能6点（フィーチャーカード、2x3グリッド）
6. CTA（ダウンロード再掲）
7. プライバシーポリシー
8. フッター

## キャッチコピー
「毎日の記録が、一生の宝物に。」

## 機能6点
1. GPS散歩記録（📍 バックグラウンド対応）
2. 10種類の記録（散歩・体重・食事・便・病院・お薬・お出かけ・日記・支出・予定）
3. 健康カルテ（🏥 病院でそのまま見せられる）
4. カレンダー管理（📅 予定・記録を日ごとに一覧）
5. 多頭飼い対応（🐕 ペット切り替え）
6. 家族で共有（クラウド同期）

## モックアップ（SVGで作成済み）
